
## React + Redux + Typescript Template Project

- Template project with react wired with redux
- Typescript with tslint configured
- Template test for components implemented with enzyme
- React Router v4
- Bootstrap 4 included

## How to get it work

- Install/Download this repository
- In the root folder, type in terminal ```npm install```
- Followed by ```npm start``` or ```npm run start```
- To run the tests, use ```npm run test```
